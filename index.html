Morse Blink

A real-time eye-blink Morse decoder built with Python, OpenCV, and MediaPipe. This Pages site hosts the project landing page and an interactive Morse demo while the webcam-powered decoder runs locally.

Run it locally Try the demo

What this project does

It detects blinks from a webcam, converts short and long blinks into Morse code, and decodes complete letters and words in real time.

GitHub Pages note

GitHub Pages only serves static files, so the live webcam stream cannot run from Pages itself. The full camera experience is still available from the local Python app.

Quick start

Install the Python dependencies, launch the local app, and keep your face visible to the camera for decoding.

Run it locally

  1. Install dependencies from the repository requirements file.
  2. Run the app with the webcam-enabled interface.
  3. Use your face and eye blinks to spell out Morse code.
pip install -r requirements.txt
python main.py --web
